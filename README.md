# OGC-Anime — 3D Character Animation

A browser-based 3D character animation built with **Three.js (WebGPU)** and the `disfigure` figure library, created for the Computer Graphics (OKG) course.

**Website:** https://siana1s.github.io/OGC-Anime/template.html

### Features
- Animated humanoid figures with looping body motion
- Music-synced "Гимнастика за релакс" workout scene
- Runs fully client-side from CDN modules
- Plain template plus a completed sound version
